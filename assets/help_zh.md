# 节拍器使用帮助

## 基本操作

- 点击中央播放按钮开始/停止节拍
- 上下拖动 BPM 圆弧边缘调节速度
- 点击 +/- 按钮微调 BPM
- 长按 +/- 按钮快速调节
- 点击 TAP 按钮测速
- 4/4 表示每小节 4 拍，四分音符为一拍

## 速度术语

| 术语 | BPM 范围 | 含义 |
|------|----------|------|
| Grave (庄板) | 25-45 BPM | 极慢 |
| Largo (广板) | 40-60 BPM | 很慢 |
| Adagio (柔板) | 55-76 BPM | 慢速 |
| Andante (行板) | 73-120 BPM | 散步速度 |
| Moderato (中板) | 108-132 BPM | 中速 |
| Allegro (快板) | 120-156 BPM | 快速 |
| Vivace (活板) | 144-176 BPM | 活泼快速 |
| Presto (急板) | 168-240 BPM | 极快 |

## 节拍乐理

- **BPM**: 每分钟节拍数
- **分子**: 每小节的拍数 (1-16)
- **分母**: 音符类型 (1=全音符, 2=二分, 4=四分, 8=八分, 16=十六分)
- **重音**: 每小节第一拍通常为重拍
- **常见拍号**:
  - 4/4 (流行音乐)
  - 3/4 (圆舞曲)
  - 6/8 (爱尔兰风)

## Tap Tempo 使用

连续点击 **TAP** 按钮 4 次，系统计算平均间隔转换为 BPM。

**常用于**：跟随音乐测速、不知道确切 BPM 时快速设置。

## 快捷键

| 按键 | 功能 |
|------|------|
| 空格键 | 播放/暂停 |
| 上箭头 | BPM +1 |
| 下箭头 | BPM -1 |
| 左上箭头 | BPM -10 |
| 右上箭头 | BPM +10 |
